# Neo4j Gaming Platform Database

## Overview

This project was completed as part of the Advanced Database Design course (Fall 2025).

The objective was to design and implement a graph database representing a gaming platform ecosystem using Neo4j. The database models relationships between players, games, achievements, leaderboards, platforms, and transactions.

## Technologies

- Neo4j
- Cypher Query Language
- Graph Database Design
- Data Modeling

## Database Components

The graph database includes:

- Players
- Games
- Achievements
- Leaderboards
- Gaming Platforms
- Transactions

## Key Features

### Graph Database Design
Designed a graph-based schema to model relationships between players, platforms, transactions, achievements, and games.

### Query Development
Developed Cypher queries for:

- Node filtering
- Relationship traversal
- Multi-hop graph analysis
- Aggregation and counting
- Ranking and leaderboard analysis
- Transaction analysis

### Example Analyses

- Retrieve player achievements
- Identify top-ranked players
- Analyze platform transactions
- Explore player-game relationships
- Generate leaderboard statistics
